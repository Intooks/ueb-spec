# Спецификация формата `.ueb`
`.ueb` (Universal Electronic Book) — это универсальный формат для электронных книг,
сторигеймов, чат-игр и визуальных новелл. Словом, любой интерактивной (но не
обязательно) литературы.

## Спецификация
`.ueb` представляет собой `.zip`-архив, хранящий всё необходимое для книги.
Структура эквивалентна рабочему проекту, поэтому для "компиляции" достаточно просто
заархивировать его. Все текстовые ресурсы должны быть в кодировке UTF-8.

Далее представлена структура файлов:
```
.
├── assets/
│   ├── images
│   ├── music
│   ├── sfx
│   └── video
├── chapters/
│   ├── 0001.uchapter
│   └── 0002.uchapter
├── resourсes/
│   └── characters/
│       ├── Я.yaml
│       └── Друг.yaml
├── scripts
├── cover.webp
├── mimetype
├── project.yaml
└── toc.yaml
```

### `mimetype`
Текстовый файл со строкой `application/ueb+zip`

### `project.yaml`
Основная мета-информация о книге. Сюда включаются как базовые для любого контента
данные, такие как название, автор и язык, так и более специализированные, например
то, как будет отображена интерактивная книга: обычными страницами, чатом, новеллой
или карточной игрой

### `toc.yaml`
Содержание книги, состоящее из названий `.uchapter`-файлов и самих глав
